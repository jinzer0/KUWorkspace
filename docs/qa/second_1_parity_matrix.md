## Clarification blockers

- Equipment add initial status: the memo and current plan need one answer for whether newly added equipment starts as `disabled` or `available`.
- Room capacity: the memo permits `1+`, seed examples and room names imply `4`, `6`, `8`, and some validation messages say error range `1..8`. Pick the authoritative admin add/edit rule and user attendee rule.
- Equipment memo `0` behavior: one section says `0` cancels during memo input but also says memo is stored as empty. Pick cancel with no write or continue with empty memo.
- Room maintenance end-date wording: the memo has wording that appears inverted around start/end and next-day 09:00. Confirm whether an admin-entered end date maps to that date at 09:00 or the following date at 09:00.
